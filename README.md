# Francesco Cerni - Personal Website

This repository contains the source code for the personal website and portfolio of Francesco Cerni, a professional working in Healthcare Innovation & Startups.

## Overview
The website is a static, responsive single-page application built with plain HTML, CSS, and minimal JavaScript (if any). It serves as a digital resume and portfolio, highlighting:
- **About**: Background in biotech, business, and pharma (Sanofi, Fidia).
- **Experience**: Timeline of professional roles and entrepreneurial projects (SCAR Holding).
- **Projects**: Showcase of healthcare innovation and early-stage startup investments.
- **Education**: Academic background.

## Structure
- `index.html`: The main HTML file containing the content and inline styling.
- `assets/`: Directory containing all image assets (logos, profile pictures, backgrounds) and videos used on the site.

## Deployment
The website is designed to be hosted directly on **GitHub Pages**.

To deploy or update:
1. Make changes to the code or assets.
2. Commit and push the changes to the `main` branch.
3. GitHub Pages will automatically rebuild and deploy the latest version.
